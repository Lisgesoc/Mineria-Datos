# Minería de Datos

## Resumen de la Asignatura

La asignatura de Minería de Datos se centra en el estudio de técnicas y algoritmos para extraer conocimiento útil a partir de grandes volúmenes de datos. El objetivo principal es aprender a identificar patrones, tendencias y relaciones ocultas en los datos que sean de utilidad para la toma de decisiones.

## Temarios de la Asignatura

### 1. Introducción a la Minería de Datos
- Conceptos básicos y definiciones
- El proceso de descubrimiento de conocimiento (KDD)
- Tipos de datos y formatos
- Aplicaciones de la minería de datos en diferentes dominios

### 2. Preprocesamiento de Datos
- Limpieza de datos: tratamiento de valores perdidos y outliers
- Integración y transformación de datos
- Normalización y estandarización
- Selección de características (feature selection)
- Reducción de datos

### 3. Técnicas de Clasificación
- Árboles de decisión
- Clasificadores bayesianos
- K-Nearest Neighbors (K-NN)
- Support Vector Machines (SVM)
- Redes neuronales
- Evaluación de modelos: matrices de confusión, precisión, recall, F1-score

### 4. Reglas de Asociación
- Algoritmo Apriori
- Conceptos de soporte, confianza y lift
- Generación de reglas de asociación
- Aplicaciones en análisis de mercado

### 5. Clustering (Agrupamiento)
- Algoritmos de particionamiento: K-means, K-medoids
- Algoritmos jerárquicos: aglomerativos y divisivos
- Clustering basado en densidad: DBSCAN
- Evaluación de la calidad del clustering
- Métricas de similitud y distancia

### 6. Reducción de Dimensionalidad
- Análisis de Componentes Principales (PCA)
- Análisis de Correspondencias
- Técnicas de selección de atributos
- Visualización de datos de alta dimensión

### 7. Evaluación y Validación de Modelos
- Partición de datos: training, validation, test
- Validación cruzada (cross-validation)
- Métricas de evaluación según el tipo de problema
- Overfitting y underfitting
- Técnicas de ensemble learning

### 8. Herramientas y Tecnologías
- Python y librerías: Pandas, NumPy, Scikit-learn
- Jupyter Notebooks
- Visualización de datos: Matplotlib, Seaborn
- Frameworks de Machine Learning

## Objetivos de Aprendizaje

- Comprender los fundamentos teóricos de la minería de datos
- Aplicar técnicas de preprocesamiento y análisis exploratorio
- Implementar y evaluar algoritmos de clasificación y clustering
- Interpretar y comunicar resultados de análisis de datos
- Desarrollar soluciones prácticas a problemas reales usando minería de datos
